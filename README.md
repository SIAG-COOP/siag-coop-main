# siag-coop-main
*SIAG COOP** es una cooperativa informática dedicada a la creación de software libre, herramientas éticas y sistemas de inteligencia autónoma orientados al bien común.   Nace del proyecto **SIAG (Sistema Integral Autónomo Grupal )**, expandido hacia una dimensión cooperativa y solidaria para promover la autogestión tecnológica.
